# Business Number Parser

Веб-приложение для сбора контактных данных заведений с Яндекс Карт.  
Парсер собирает названия, телефоны, адреса и категории организаций по городам и ключевым словам.

## Возможности

- **Поиск по базе данных** — мгновенный поиск по предварительно собранным данным
- **Ручной поиск** — парсинг Яндекс Карт в реальном времени по произвольным ключевым словам
- **Прокрутка результатов** — автоматическая прокрутка сайдбара + перехват AJAX-ответов для сбора всех результатов (не только первых 25)
- **Экспорт в Excel** — выгрузка результатов в `.xlsx`
- **23 города** и 15 категорий из коробки

## Технологии

| Слой | Стек |
|------|------|
| Frontend | Vue 3, Vite |
| Backend | Express.js |
| Парсер | Playwright (Chromium) |
| Экспорт | ExcelJS |

## Установка

```bash
git clone <repo-url>
cd business-number-parser
npm install
npx playwright install chromium
```

## Сбор данных (парсер)

Парсер открывает Chromium, загружает страницы Яндекс Карт, извлекает данные из встроенного JSON и AJAX-ответов при прокрутке. Результаты сохраняются в `data/db.json`.

```bash
# Все города и категории
npm run scrape

# Один город и категория
npm run scrape -- --city moscow --category кафе

# Пересобрать (игнорировать кэш)
npm run scrape -- --city sochi --force true
```

При появлении CAPTCHA браузер остаётся открытым — решите её вручную, парсер продолжит автоматически.

## Запуск

```bash
# Разработка (Vite + Express)
npm run dev

# Продакшен
npm run build
npm start
```

Откройте `http://localhost:5173` (dev) или `http://localhost:3001` (production).

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/cities` | Список городов |
| GET | `/api/categories?city=moscow` | Категории для города |
| GET | `/api/search?category=кафе&city=moscow` | Поиск по базе |
| GET | `/api/live-search?query=пиццерия&city=moscow` | Парсинг с Яндекс Карт в реальном времени |
